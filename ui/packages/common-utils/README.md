## 사용하기

1. [`Mr.C/ui` 프로젝트간 참조](../../README.md#프로젝트간-참조) 에 따라 설정합니다.

2. 사용 예시

   ```typescript
   import { hello } from '@mrc/common-utils';

   const message = hello('world');

   console.log(message); // Hello, world!
   ```
